Datatypes in Javascript

  • Number-A number data type can be an integer,
    a floating point value, an exponential value, a ‘NaN’ or a ‘Infinity’.
  • String - A string can be any group of characters enclosed by a single or
    double-quotes or by backticks.
  • Boolean- The boolean data type has only two values, true and false.
    It is mostly used to check a logical condition. once declared its value
    cannot be changed
  • Undefined- Undefined data type means a variable that is not defined.
    The variable is declared but doesn’t contain any value.
  • Null - Null is a data type that is represented by only one value, the ‘null’ itself.
    A null value means no value.Front End Web Development Training
  • Symbol - The symbol data type defines a property of an object
    which is private to the object.
  • Array- An array contains more than one value with a numerical index,
    where the index starts from 0. Thus it holds its value in a key-value pair.
  • Object- An Object contains key-value pairs in its address
